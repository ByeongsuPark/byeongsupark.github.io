# 로컬 서버 액세스

안드로이드 개발을 할 때, 대부분의 경우는 서버와 통신을 하는 기능이
구현된다.  
   나같은 경우는 개발, 테스트 서버가 따로 없다.  
   그래서 서버 클라이언트(안드로이드) 개발을 모두 로컬에서 한다.  
   그런데 이게 여간 귀찮은 게 아니다.   
   항상 로컬에서 서버 돌리면 클라이언트 단에 서버 주소를
   제공해야 하기 때문이다.  
그래서 이번 포스팅에서는 안드로이드 기기에서 로컬 서버에 접근하는 보다 쉬운 방법을 소개
할려 한다.
